# E-commerce React + Firebase

## Descripción

SPA de e-commerce desarrollada con React + Vite. Incluye catálogo, categorías, detalle, carrito y checkout.

## Tecnologías

- React
- React Router
- Firebase Firestore

## Funcionalidades

- Listado y detalle de productos desde Firestore
- Filtrado por categoría
- Carrito global con Context
- Checkout con creación de orden en Firestore y devolución de ID

## Instalación

1. Clonar repo
2. `npm install`
3. Crear `.env` con credenciales Firebase (VITE\_...)
4. `npm run dev`
